# SmartTax - A Self-Assessment Tool for UK Freelancers

**Author:** Tunde Debreczeni
**Course:** BCS (Hons) Applied Computing, University of Wales Trinity Saint David

## System Architecture


1. **Frontend**
   - Provides a progressive and user-friendly dashboard interface.
2. **Backend API** 
   - Handles business logic, data validation, and tax calculations.
3. **Database**
   - Stores user data, income, expenses, and tax records.


## User Features

- User-friendly interface for freelancers to manage income and expenses.
- Automatic **tax and National Insurance estimations** based on UK thresholds.
- Integration-ready for **HMRC's Making Tax Digital** API.
